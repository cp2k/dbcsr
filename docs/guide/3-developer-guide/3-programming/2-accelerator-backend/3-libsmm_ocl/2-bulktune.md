# Parameters

Bulk tuning and tuned-parameter management for OpenCL SMM kernels are now part of the LIBXSTREAM SMM sample rather than DBCSR's documentation. See [LIBXSTREAM](https://libxstream.readthedocs.io/) for the current scripts, partitioning options, and tuned-parameter update workflow.

DBCSR users normally only need the resulting LIBXSTREAM/LIBXS installation and the DBCSR build option `-DUSE_ACCEL=opencl`.
